2、编写程序，统计1+3+5+……+99，结果存入DX寄存器中。

程序如下:

```assembly
assume cs:code1
code1 segment
          mov  cx,99
          mov  dx,0
    s:    add  dx,cx
          loop s
          mov  ax,4c00h
          int  21H
code1 ends
end

```

3、将0FFFE0\~0FFFFFH中的16个字，顺序复制到00200H\~0021FH中，

程序如下：

```assembly
assume cs:code1
code1 segment
          mov  ax,0FFFEH
          mov  ds,ax
          mov  ax,0020H
          mov  es,ax
          mov  bx,0
          mov  cx,16
    s:    mov  ax,ds:[bx]
          mov  es:[bx],ax
          add  bx,2
          loop s
          mov  ax,4c00H
          int  21H
code1 ends
end
```
